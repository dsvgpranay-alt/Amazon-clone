<h1>Amazon Clone</h1> 

This project is a frontend clone of the Amazon website built using HTML and CSS. It focuses on replicating the user interface and layout of a modern e-commerce platform.

<h2>Features</h2>
<ul>
  <li>Responsive homepage layout</li>
  <li>Navigation bar with search UI</li>
  <li>Product sections and cards</li>
  <li>Banner and promotional sections</li>
  <li>Footer similar to Amazon</li>
  <li>Clean and structured UI design</li>
</ul>
<h2>Tech Stack</h2>
<ul>
  <li>HTML5</li>
  <li>CSS</li>
</ul>
